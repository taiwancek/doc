廣義推播定義 ¶

類型 ¶

  • PUSH: 代表android使用FCM推播，ios使用APNS推播

  • MQTT: 代表透過MQTT推播

行為 ¶

  • 帳號加入群組

    • PUSH > 群組使用者手機
    • MQTT > 群組裝置

  • 群組發布訊息

    • PUSH > 群組使用者手機

  • 群組觸發行為

    • PUSH > 群組使用者手機
    • MQTT > 群組裝置

  • 群組解散

    • PUSH > 群組使用者手機
    • MQTT > 群組裝置

行為格式 ¶

  • 推播類型(type欄位)
join_group: 成員加入群組
dissolve_group: 解散群組
publish_msg: 發佈訊息
trigger_behavior: 觸發裝置行為
behavior_return: 裝置行為處理狀態

  • 成員加入群組

    • data結構

      "group":{
          "uuid":"458f338b-dd93-46e8-a931-e687eae111111",

      "inviter":{//邀請者
          "uuid":"458f338b-dd93-46e8-a931-e687eae4f5ce",
          "name":"kevin"
      },
      "member":{//新加入成員
          "uuid":"458f338b-dd93-46e8-a931-e687eae4f5ce",
          "name":"Jib",
          "is_bot":false //true代表此成員為IOT裝置
      }
  }
    • Android
    {
      "notification" : {

              "title":"Jib已加入『路竹好好玩』群組"
