Desktop所需API ¶

  1. 登入 | /v2/api/user/login

  2. 取得材料列表 | /api/material/list?enable=true

  3. 取得當日申請序號資訊 | /api/prod_record/get

  4. 新增產品序號記錄 | /api/prod_record/create

  5. 取得單筆產品序號結果 | /api/prod_product_sn/get

  6. 修改單筆產品序號結果 | /api/prod_product_sn/update

DB Schema ¶

migrate start number: 00043

CREATE TABLE material ( 
    id                   bigserial  NOT NULL,
    user_uuid            uuid  NOT NULL,
    sn                   varchar(64)  NOT NULL,
    vendor_id            varchar(64)  NOT NULL,
    product_id           varchar(64)  NOT NULL,
    vendor_name          varchar(64)  NOT NULL,
    product_name         varchar(64)  NOT NULL,
    model_number         varchar(64)  NOT NULL,
    serial_number        varchar(64)  NOT NULL,
    sn_prefix            varchar(4)  NOT NULL,
    enable               bool DEFAULT true NOT NULL,
    created_at           timestamptz DEFAULT current_timestamp NOT NULL,
    updated_at           timestamptz DEFAULT current_timestamp NOT NULL,
    CONSTRAINT pk_material_id PRIMARY KEY ( id )
 );

COMMENT ON TABLE material IS '材料表';
COMMENT ON COLUMN material.sn IS '料號';
COMMENT ON COLUMN material.sn_prefix IS '料號前綴英文,用來寫入裝置';


================================================
CREATE TABLE prod_record ( 
    id                   bigserial  NOT NULL,
    user_uuid            uuid  NOT NULL,
    start                integer  NOT NULL,
    num                  integer  NOT NULL,
    created_at           timestamptz DEFAULT current_timestamp NOT NULL,
    CONSTRAINT pk_prod_record_id PRIMARY KEY ( id )
 );

COMMENT ON TABLE prod_record IS '產品序號申請記錄表';
COMMENT ON COLUMN prod_record.start IS '開始編號，每天都會歸零';
COMMENT ON COLUMN prod_record.num IS '申請生產數量';

================================================
CREATE TABLE prod_product_sn ( 
    id                   bigserial  NOT NULL,
    sn_id                bigint  NOT NULL,
    p_record_id          bigint  NOT NULL,
    udid                 varchar(32)  NOT NULL,
    result               smallint DEFAULT 0 NOT NULL,
    created_at           timestamptz DEFAULT current_timestamp NOT NULL,
    updated_at           timestamptz DEFAULT current_timestamp NOT NULL,
    CONSTRAINT pk_prod_product_sn_id PRIMARY KEY ( id )
 );

COMMENT ON TABLE prod_product_sn IS '產品序號表';
COMMENT ON COLUMN prod_product_sn.sn_id IS '料號ID';
COMMENT ON COLUMN prod_product_sn.p_record_id IS '產品序號申請ID';
COMMENT ON COLUMN prod_product_sn.udid IS '料號前綴英文+日期6碼(西元年月日，例如180628)+序號5碼';
COMMENT ON COLUMN prod_product_sn.result IS '格式: 0:未回傳結果(預設)|1:產品序號寫入成功|2:產品序號寫入失敗';

Login for third
POST/api/user/login_third

第三方登入

  • 若第一次使用第三方登入，將會自動在本地端建立一組帳號(email)密碼與第三方登資訊綁定。 例如第一次使用facebook登入，將會取得一組Fackebook Id與Name,這時系統將會自動建立本地端帳號({Id}@facebook)與密碼(16字元亂數)並且綁定。
